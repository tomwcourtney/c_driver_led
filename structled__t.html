Detailed Description

Holds state information for an led's configuration.

Parameters
id- A unique identifier for an led.
enabled- True if led is in use, false otherwise.
pinout- A pinout object that defines connection of the led to the microcontroller.
sequence_id- A pointer to a sequence object that defines the flash pattern of the led.
sequence_idx- Maintains the position in the assigned sequence that the LED is up to
timer_count- Maintains the acrued time since the last sequence index increment
